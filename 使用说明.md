# Cody + WXCody 联合使用说明

本文档面向 **WXCody 小程序**用户，并简述 **Cody 设备**配合操作。技术细节见本目录 **`README.md`**；固件烧录与开发见 **Cody** 工程中的 **`README.md`**（与本书同级目录时多为 `../Cody/README.md`）。

---

## 1. 使用前

- 手机打开 **蓝牙**，允许 **微信** 使用蓝牙；Cody **开机**；手机靠近设备。  
- 蓝牙设备名一般为 **`Cody-` + 后缀**（以扫描列表为准）。

---

## 2. 连接与配对

### 小程序侧

- **闪屏**：已连接则进控制台；有上次设备则尝试**自动回连**；否则约 3 秒内失败则进**连接页**。  
- **连接页**：首次进入本会话会**自动扫一次**；若因拒绝/断线再回到连接页，需点 **「扫描附近的 Cody」**。点列表项连接；首次或未信任时可能弹窗，需在 **Cody 屏幕**上确认。  
- **断线**：一般会 **reLaunch 回连接页**，避免假在线。

### 设备侧（首次或未信任手机）

> 设备顶部有两个按钮，左键为设备的操作键，右键为设备重启键。

- 屏幕出现 **连接确认**（显示小程序上报的**名称**，用于确认是哪台手机）。  
- **短按左键**确认键约 **0.6 s 内松开**：**拒绝**，蓝牙断开。  
- **长按左键**约 **1.2 s**（进度走完）：**允许**。  
- 已绑定的手机常**自动放行**；换机需重新确认或在小程序侧清除信任后再绑。


---

## 3. 小程序里怎么用

**控制台**底部：**模式切换** | **系统设置**。

- **模式**：点名称切换设备模式；点 **⚙️** 进独立页（已支持：**图片 / 笔记 / 手绘**；时钟、表情若无独立页会提示未开放）。  
- **系统设置** → **连接**（断开、清信任相关）、**存储**（用量；格式化/恢复出厂，需确认）、**更新**（检查版本、在线升级、**本地选 `firmware.bin`**）、**背光**。  
- **图片**：轮播与图槽上传/替换；缩略图从设备拉取，**断连会丢进度，重连再同步**。  
- **笔记**：轮播、增删改置顶（单条最多约 100 字，以界面为准）。  
- **手绘**：画板同步有延迟属正常；清屏、颜色、粗细（2～16 px）、镜像、「你画我猜」；切后台会尽量通知设备结束游戏展示。

**本地固件升级**：用微信 **聊天文件** 选 `.bin`；系统目录里的文件可先发到 **文件传输助手** 再选。

---

## 4. 设备按键（无菜单时 / 在设置菜单内）

| 场景 | 短按 | 长按约 1.2 s |
|------|------|----------------|
| 正常显示 | 循环切换显示模式 | 进 **系统设置** |
| 设置菜单内 | 切换菜单项 | 确认/进入（以屏上提示为准） |

图片/笔记/内容管理多在小程序完成；设备负责显示与本地菜单。

## 5. 模式一览

| 模式 | 要点 |
|------|------|
| 图片 | 图槽、轮播 |
| 时钟 | 时间；可经小程序同步手机时间 |
| 笔记 | 内容与轮播 |
| 表情 | 动画展示 |
| 手绘 | 与手机同步；断连会收掉你画我猜等状态 |

---

## 6. 存储、恢复出厂与 OTA

- **危险操作**仅在确认后执行；完成后常**重启**、**断蓝牙**，需重新连接。  
- **OTA**：在「更新设置」在线或本地升级；过程中勿远离、勿断电；结束后面板重启属正常。  
- 固件内 **WiFi/Web 调试** 默认 **关闭**（`CODY_ENABLE_WIFI_DEBUG`），一般用户无需开。

---

## 7. 常见问题

| 现象 | 建议 |
|------|------|
| 连不上 / 总回连接页 | 查蓝牙与权限、重启设备与蓝牙、连接页手动扫描 |
| 提示连接被拒绝 | Cody 上长按允许；或小程序取消后重试 |
| 换手机不能控 | 设备上重新配对确认，或小程序「断开/清信任」流程后再连 |
| 缩略图一直转 | 保持连接；重进图片页或重连后再拉 |
| 升级异常 / 黑屏 | 勿断电；错误包需按 **Cody 固件 README** 重新烧录 |
