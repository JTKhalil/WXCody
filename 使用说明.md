# WXCody 小程序使用说明

本文档依据当前工程内页面与逻辑整理，便于用户按界面完成连接、模式管理与系统设置。小程序导航栏标题多为「WXCody」，控制台页标题为「Cody 控制台」。

---

## 一、使用前准备

1. **打开手机蓝牙**，并允许微信使用蓝牙权限。  
2. **Cody 设备开机**，并处于可被发现状态（广播名称一般为 `Cody-` 前缀，具体以扫描列表为准）。  
3. 在**较近距离**内使用，避免强干扰环境，以减少蓝牙断连。

---

## 二、启动与连接

### 1. 闪屏页

- 打开小程序后首先进入**闪屏页**。  
- 若**已与 Cody 保持蓝牙连接**（例如刚从后台返回），会直接进入**控制台**。  
- 若本地**保存过上次连接的设备**，会尝试在几秒内**自动回连**；成功且设备端已信任，则进入**控制台**。  
- 若无历史设备、自动连接失败或超时（约 3 秒兜底），会进入**连接页**。  
- 闪屏过程中可能短暂显示「正在连接…」等状态。

### 2. 连接页

- **当前设备名称**：展示本机在配对流程中使用的显示名（用于 Cody 端确认是谁在连接）。  
- **扫描附近的 Cody**：手动扫描周边设备；列表中出现以 `Cody-` 等为名的条目。  
- **点击设备名称**发起连接：  
  - **首次或未信任**：可能出现弹窗，提示在 **Cody 屏幕**上确认连接；也可点「×」关闭弹窗并断开本次连接。  
  - **已信任设备**：通常可直接进入控制台。  
- **扫描策略（与代码一致）**：在同一次打开小程序的过程中，**仅在第一次进入连接页时自动扫描一次**；若因拒绝连接、断线等原因再次回到连接页，**不会自动再扫**，需要手动点「扫描附近的 Cody」。  
- 连接过程中请留意系统蓝牙是否关闭；若提示与蓝牙相关，请先打开手机蓝牙。

### 3. 断线后的行为

- 蓝牙断开时，小程序会**统一跳回连接页**（避免在控制台等页面误以为仍在线）。  
- 回到连接页后按上一条策略：**不自动扫描**，需手动扫描或依赖闪屏时的自动回连逻辑（视入口而定）。

---

## 三、控制台（Cody 控制台）

底部有两个主 Tab：**模式切换**、**系统设置**。

### 1. 模式切换（Tab「模式切换」）

- 展示五种模式：**图片 / 时钟 / 笔记 / 表情 / 手绘**。  
- **点模式名称**：向设备下发切换模式指令（当前高亮表示设备当前模式，会与设备侧同步刷新）。  
- **点右侧「⚙️」**：进入该模式的**独立设置页**（见下文）；其中**时钟、表情**在代码中若未配置独立页，会提示「功能还未开放」。  
- 实际已接入的独立页为：  
  - 图片 → **图片模式**  
  - 笔记 → **笔记模式**  
  - 手绘 → **手绘模式**

### 2. 系统设置（Tab「系统设置」）

入口为列表，点击后进入子页面（需保持蓝牙连接，部分操作在断线时不可用）：

| 入口       | 说明 |
|------------|------|
| **连接设置** | 查看连接状态、当前 Cody 名称、本机显示名；可**断开与 Cody 的连接**（会清本地缓存与信任关系等）。 |
| **存储空间** | 查看设备端总空间 / 已用 / 可用；进入页面或重连后会自动拉取。**危险操作**：删除 Cody 上所有数据、恢复出厂设置（均需二次确认）。 |
| **更新设置** | 查看当前固件版本、**刷新**检查更新、有新版时**升级**（从配置的远端下载固件）；也可**选择本地 firmware.bin 升级**（从微信聊天文件中选取 `.bin`，推送到设备）。升级过程中控制台顶部可能显示进度；完成后设备通常会重启并断连。 |
| **背光亮度** | 0～255 滑动调节设备背光。 |

### 3. 固件升级进度

- 若在后台进行固件升级，控制台**顶部**可能出现**进度条**，用于展示整体进度。

---

## 四、图片模式（页面标题：图片模式）

- **图片轮播**：开关「启用轮播」；**间隔** 3～60 秒可调。  
- **图槽**：每槽可「上传」或「替换」图片；上传中可能显示进度，**推送中**可「取消上传」。  
- **缩略图**：设备端已有图时，会从设备**拉取缩略图**显示；若传输中断或断连，进度会重置，**再次连接后会重新同步**（无需旧进度续传）。  
- 点缩略图区域上的「×」可删除该槽图片（具体以设备与协议为准）。

---

## 五、笔记模式（页面标题：笔记模式）

- **笔记轮播**：开关与 **3～60 秒**间隔。  
- **新增 / 编辑笔记**：最多 **100 字**；保存需保持连接。  
- **历史笔记**：可**置顶**、**编辑**、**删除**（部分操作需连接）。

---

## 六、手绘模式（页面标题：手绘模式）

- **画板**：手指绘制，笔迹会经蓝牙同步到 Cody；**存在一定延迟属正常现象**。  
- **清屏**：清空画板（有遮罩提示时可能暂时不可操作）。  
- **画笔颜色**：点选色块。  
- **画笔粗细**：**2～16 px**，步进 2（默认较细，便于精细绘制）。  
- **镜像模式**：开关用于左右镜像绘制效果。  
- **你画我猜**：开始游戏后按提示词绘画；可**结束游戏**或**跳过**；游戏进行中可能有全屏遮挡提示。  
- **从设备同步画稿**：进入页面等情况可能出现「正在从设备同步画稿」与百分比进度。  
- **小程序切到后台**：若「你画我猜」正在进行，程序会在后台尝试通知设备结束相关展示。

---

## 七、常见问题提示

1. **一直进连接页**：检查蓝牙、距离、Cody 是否已开机；在连接页**手动扫描**并重新点设备连接。  
2. **提示连接被拒绝**：在 Cody 上确认配对；或稍后再试；已信任设备若频繁拒绝，可检查设备是否重启、忙线。  
3. **断连频繁**：避免多开占蓝牙的应用；关闭省电限制；仍异常时可结合小程序内**日志**（若页面提供）或开发者工具排查。  
4. **图片缩略图一直转圈**：确认连接稳定；断开过则重新进入页面或等待自动重新拉取；必要时在图片模式重新触发同步。  
5. **清空数据与恢复出厂**：均在「存储空间」页，务必确认后再操作；恢复出厂可能导致设备重启并断连。

## 八、本地固件文件说明

- 「选择本地 firmware.bin 升级」使用微信的 **聊天文件** 选择能力（`chooseMessageFile`）。若手机里仅有系统目录下的 `firmware.bin`，可先将其**发送到微信「文件传输助手」**，再在升级页中选择该聊天文件。
